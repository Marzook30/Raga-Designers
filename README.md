# Raga Designers Landing Page Redesign

## Overview

This project is a UI/UX redesign of the Raga Designers landing page created as part of a technical assessment. The objective was to improve the overall user experience by creating a clean, modern, and responsive interface while maintaining the brand's identity.

## Features

* Modern and responsive landing page
* Improved visual hierarchy
* Clear call-to-action sections
* Mobile-friendly design
* Clean typography and spacing
* Optimized user experience

## Technologies Used

* HTML5
* CSS3
* JavaScript
